# 脚本框架模板

适用于短视频（抖音/视频号/TikTok/YT Shorts）。图文/长文可只取「钩子—价值—转化」三段。

## 结构

1. **钩子（0-3 秒）**
   - 目标：3 秒内让目标人群"这跟我有关"。
   - 手法：抛痛点 / 制造反差 / 蹭热点事件 / 抛结论。
   - 台词：{{一句话}}

2. **建立相关（3-8 秒）**
   - 点明这条内容能解决的具体问题或带来的好处。
   - 台词：{{一句话}}

3. **正文价值（8-40 秒）**
   - 分点给"干货/演示/故事"，节奏紧凑，每 5-8 秒一个信息点。
   - 关键：自然植入产品能力（如 AI 毛绒玩具"能干活"的真实演示），不硬广。
   - 要点：
     - {{要点1}}
     - {{要点2}}
     - {{要点3}}

4. **转化 / CTA（结尾）**
   - 关联卖点，给一个明确动作（关注/评论关键词/主页了解/私信）。
   - 台词：{{一句话}}

## 拍摄/呈现建议
- 形式：{{口播 / 演示 / 剧情 / 图文卡片}}
- 时长：{{建议时长}}
- 字幕/封面钩子：{{封面大字}}

## 备注
- 保持账号"人味儿"与人设一致，避免明显 AI 味的空话。
- 数据/功效表述需真实可验证，规避夸大承诺。
