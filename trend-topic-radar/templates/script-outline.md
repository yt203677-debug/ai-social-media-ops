# 脚本框架库

产出选题时，从下面选**一个**最合适的框架套用，并把该框架的每一步都填成**具体台词**（不要只写结构名）。
适用于短视频（抖音/视频号/TikTok/YT Shorts）；图文/长文可只取「钩子 → 价值 → 转化」三段。

## 怎么选框架
- 讲"解决一个具体问题" → 框架A 痛点解决型(PAS)
- 有反常识结论/想制造惊讶 → 框架B 反转型
- "N 个/几种/盘点" → 框架C 盘点清单型
- 产品自证/情感种草 → 框架D 故事种草型

---

## 框架A · 痛点解决型（PAS：痛点 → 放大 → 方案 → 行动）
1. **痛点(0-3s)**：直接戳中人群的具体烦恼。
2. **放大(3-10s)**：说清这个痛点不解决的代价/后果，让人"对号入座"。
3. **方案(10-40s)**：给出方法，分 2-3 点，自然带出产品能力。
4. **行动(结尾)**：一个明确动作。

> 迷你示例（AI 玩具方向）：
> - 痛点：买的 AI 陪伴玩具，聊两天就腻了？
> - 放大：因为它只会陪你尬聊，聊得越久越像在完成任务，最后吃灰。
> - 方案：换个思路——让玩具帮你干活：①早上给你选题 ②帮你改多平台文案 ③随手查资料。
> - 行动：想看它怎么干活，主页找我。

## 框架B · 反转型（黄金3秒钩子 → 建立预期 → 中间反转 → 结尾转化）
1. **钩子(0-3s)**：抛一个吸睛结论或悬念。
2. **建立预期(3-8s)**：顺着大家的常识往下说。
3. **反转(8-30s)**：打破预期，给出反常识的真相/做法。
4. **转化(结尾)**：把反转落到产品卖点 + CTA。

> 迷你示例：
> - 钩子：AI 陪伴玩具今天起要防沉迷了，但有一种玩具根本不怕。
> - 建立预期：你以为防沉迷是坏消息……
> - 反转：其实被打击的是"越聊越久才赚钱"的陪聊玩具；而帮你干活的玩具，用得越快越好。
> - 转化：一个想黏住你，一个想帮你省时间——你选哪种？主页了解。

## 框架C · 盘点清单型（钩子 → 逐条 → 转化）
1. **钩子(0-3s)**：给出数量+价值，如"3 个……90% 的人不知道"。
2. **逐条(每条 5-8s)**：每条一个信息点，节奏紧凑，可上字幕序号。
3. **转化(结尾)**：收束 + CTA。

> 迷你示例：
> - 钩子：普通人用 AI 搞钱，这 3 个方向今年最稳。
> - 逐条：①…… ②…… ③（自然带出云托管/玩具）……
> - 转化：想要一套现成的工具，主页自取。

## 框架D · 故事种草型（情境 → 冲突 → 转折 → 升华/CTA）
1. **情境(0-5s)**：一个真实场景/第一人称开场。
2. **冲突(5-15s)**：遇到的问题或反差。
3. **转折(15-40s)**：产品如何改变结果（真实演示优先）。
4. **升华/CTA(结尾)**：情绪落点 + 动作。

> 迷你示例：
> - 情境：昨晚我没写一个字，今天要发的选题全好了。
> - 冲突：以前每天光想发什么就耗掉两小时。
> - 转折：现在这只毛绒玩具(装了 OpenClaw)半夜帮我盯热点、出了 5 条选题。
> - 升华/CTA：能干活的玩具长这样，想要的评论区扣 1。

## 黄金 3 秒钩子公式库（任选其一，填成具体台词）
- 痛点直击："还在为 X 发愁吗？"
- 反差/悬念："X 今天变天了，但有一种情况例外——"
- 抛结论："关于 X，90% 的人都做错了。"
- 数字清单："3 个 X，最后一个最狠。"
- 蹭事件："就在今天，X 正式生效/发布。"
- 身份代入："如果你也在做 X，这条一定要看。"
- 结果前置："我没花一分钱，用 X 做出了 Y。"

## CTA 公式库（结尾任选其一）
- 关注类："关注我，每天一个能落地的 AI 玩法。"
- 评论钩子："想要 X 的，评论区扣『要』。"
- 主页/私信："完整方法放主页了 / 私信回复『X』领取。"
- 低门槛试用："想试的，主页有免费版。"

## 字数 ↔ 时长换算（中文口播）
- 约 4-5 字/秒。15s≈70字、30s≈130字、45s≈200字、60s≈260字。
- 产出脚本时按目标时长控制字数，别超。

## 备注
- 选定的框架每一步都要写成**能直接念的台词**，不要留结构名占位。
- 保持账号"人味儿"与人设一致，避免明显 AI 味的空话。
- 数据/功效表述需真实可验证，规避夸大承诺。
